# SevenHouse 酒吧点餐系统  H5小程序端

## 运行

> 下载HBuilder X导入项目 --> 打开pages/index/index.vue --> 头部菜单栏，点击运行 --> 选择运行到浏览器 

## 环境依赖
运行之前需要安装 nodejs 环境
